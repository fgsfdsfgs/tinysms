# TinySMS
A tiny Win32 demo that recreates the JPN Sega Master System BIOS' attract mode.

# Requirements
* Visual C++ 2012 or newer (cl.exe and link.exe);
* UPX.

# Compiling
1. Ensure cl.exe, link.exe are in PATH.
2. Run build.bat.
